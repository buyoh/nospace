# Keywords and Operators

## Declarations

- `func`
- `let`

## Flow Control

- `if`, `elsif`, `else`
- `while`
- `return`

## Operators

### Arithmetic

- `+`, `-`, `*`, `/`, `%`
- `+=`, `-=`, `*=`, `/=`, `%=`
- `-`

### Assignment

- `=`

### Comparisons

- `==`, `!=`, `<`, `<=`, `>`, `>=`

### Logical Operators

- `&&`, `||`
- `!`

### Reference and Dereference

- `&`
- `*`

## Built-in Functions

- `__puti(val)`, `__putc(val)`
- `__geti()`, `__getc()`
- `__getiv(addr)`, `__getcv()`
